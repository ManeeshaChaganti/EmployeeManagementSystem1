# Employee Management System

## 📌 Overview
Employee Management System is a Java application developed using Hibernate ORM. It provides an easy way to manage employee records with CRUD (Create, Read, Update, Delete) operations while storing data in a MySQL database.

## ✨ Features
- Add Employee
- View Employee Details
- Update Employee Information
- Delete Employee
- Hibernate ORM Integration
- MySQL Database Connectivity

## 🛠️ Technologies Used
- Java
- Hibernate ORM
- MySQL
- Maven
- Eclipse IDE
- JDBC

## 📂 Project Structure
```
src/
 ├── entity/
 ├── dao/
 ├── util/
 ├── main/
 └── resources/
```

## ⚙️ Prerequisites
- Java JDK 8 or above
- Eclipse IDE
- MySQL Server
- Maven

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/ManeeshaChaganti/EmployeeManagementSystem1.git
   ```

2. Import the project into Eclipse as a Maven project.

3. Create a MySQL database.

4. Update the database credentials in `hibernate.cfg.xml`.

5. Run the main Java class.
