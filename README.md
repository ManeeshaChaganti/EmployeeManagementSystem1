# Employee Management System

## Overview
A console-based Employee Management System developed using Java, Hibernate ORM, Maven, and MySQL.

## Technologies Used
- Java
- Hibernate ORM
- Maven
- MySQL
- Eclipse IDE

## Features
- Add Employee
- View All Employees
- Search Employee by ID
- Update Employee Salary
- Delete Employee

## Database
- MySQL
